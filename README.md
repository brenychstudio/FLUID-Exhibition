# FLUID Exhibition

Collaborative exhibition website and digital publication layer created for Santlluc Cercle Artístic, designed to present the exhibition with a premium editorial web experience.

## Overview
FLUID Exhibition is a cultural web project built as a collaborative exhibition platform and digital publication surface.
It combines editorial presentation, exhibition information, and public-facing event communication in one refined front-end system.

## Stack
- Vite
- React
- TypeScript
- Tailwind CSS
- Editorial front-end structure

- ## What it demonstrates
- Cultural/exhibition web direction
- Editorial front-end systems
- Public-facing event communication
- Premium presentation for artistic institutions
